Features

We use multiple modern data visualization techniques to animate data and use machine learning to analyze the data.

Visualization

We create modern data visualization tools to animate the gravity wave data and use machine learning to reduce the dimensional complexity.

Data Portal

Standford University and a research company built a modern data portal which enables interactive manipulation of the data.

Google Cloud

We use google Cloud to do super-computing work, create and store data in two formats. Data will be shared upon request.

Open Source

Our open-source codes and relevant simulation tools are shared in the CSSI-GW github portal, under standard software license.
